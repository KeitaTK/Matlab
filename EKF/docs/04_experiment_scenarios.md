# 実験シナリオ定義

## シナリオ一覧

### S1: ベースライン単一周波数
- 信号: 単一正弦波（必要なら減衰あり）
- 真値周波数: 0.8 Hz
- 初期推定: 0.6 Hz
- ノイズ: なし
- 干渉: なし

### S2: ノイズスイープ
- 基本信号: S1
- ノイズ標準偏差: [0.01, 0.05, 0.10, 0.20]
- 目的: バイアスと分散の劣化傾向確認

### S3: 初期周波数ミスマッチ
- 基本信号: S1
- 初期値: [0.4, 0.5, 0.6, 0.7, 1.0, 1.2] Hz
- 目的: 収束域と過渡応答の確認

### S4: 時変周波数
- 真値周波数スケジュール:
  - 区間1: 0.8 Hz
  - 区間2: 0.9 Hz
  - 区間3: 0.75 Hz
- 目的: 追従性と遅れ評価

### S5: 高周波干渉（固定）
- ターゲット: 0.8 Hz
- 干渉: 5.0 Hz
- 干渉振幅比: [0.3, 0.5]
- 任意ノイズ: 0.01
- 目的: 帯域外妨害への耐性確認

### S6: 高周波干渉スイープ
- 干渉周波数: [3, 5, 8, 12] Hz
- 干渉振幅比: [0.1, 0.3, 0.5]
- 目的: 脆弱条件の特定

## ファイル命名規則
入力信号:
- `data/sig_{scenario}_f{target}_fi{interf}_n{noise}_seed{seed}.csv`

出力結果:
- `results/res_{scenario}_Q{qtag}_R{rtag}_{timestamp}.mat`
- `results/res_{scenario}_Q{qtag}_R{rtag}_{timestamp}.csv`
- `results/fig_{scenario}_Q{qtag}_R{rtag}_{timestamp}.png`

## 推奨プロット
- 周波数追従
- 周波数誤差
- 再構成信号
- イノベーション
- 共分散対角成分
