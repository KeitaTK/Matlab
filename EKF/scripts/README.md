# scripts ディレクトリ

このディレクトリには、正弦波モデルEKFの実装スクリプトを配置します。

## 実装済み
- `ekf_harmonic_step.m`: EKF 1ステップ更新
- `generate_signal_baseline.m`: ベースライン信号生成
- `generate_signal_noisy.m`: ノイズ付与信号生成
- `generate_signal_highfreq_interference.m`: 高周波干渉信号生成
- `run_ekf_case.m`: 単一ケース実行
- `run_ekf_sweep.m`: ケース一括実行
- `evaluate_ekf_metrics.m`: 指標評価
- `plot_ekf_results.m`: 可視化
- `run_demo_harmonic_ekf.m`: デモ実行
